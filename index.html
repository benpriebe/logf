logf

A javascript logging framework for the browser with pretty colors, timestamps and filtering.

This is a demo page that allows you to see logf in action. To see the logf output you need to open up the browsers dev. tools console (usually F12 then navigate to the console tab).

logf is written to be useful and as such requires either Google Chrome or one of the other browsers with firebug or firebug light installed.

To use the filter enter in any valid regex (or plain text) to filter log messages on. If the log message contains the filter expression, it'll show up on the console otherwise it won't.

Use %o in your message to render the included object a different way.
